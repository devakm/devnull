## Commit authorship

Claude Code is a tool, not a co-author. **NEVER** add a `Co-Authored-By: Claude ...`
trailer or a `Generated with Claude Code` line to any commit message or pull-request
description. A `commit-msg` git hook strips these automatically as a backstop, but do
not write them in the first place.
