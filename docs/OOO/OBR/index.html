About this site

Every alpha / beta / release build of Oscuro's Oblivion Overhaul Remastered FULL (OOORF) is tracked in the OblivionRemastered_OOO repo, with a per-release manifest of every file (SHA-256 + size) so you can see exactly what changed between any two builds. Install archives are published as GitHub Releases.

Note: OOORF (this site) is the FULL edition. The acronym OOORS refers to the separate Slim edition and is not used here.
