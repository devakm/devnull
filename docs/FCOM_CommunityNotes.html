FCOM Community Notes

Updates, Clarifications and Community Findings for FCOM

Notes compiled by dedalo4373 (circa 2020)

This page contains community-contributed notes, updates, and clarifications for FCOM Convergence. These notes were compiled to help users navigate changes since the original documentation was written, and to document patches and fixes released by the community.

1. Required ESP Updates After Download

After downloading all the archives from TESAlliance, as an end user you need to actually update some of the .esps. These are the links:

  • FCOM Convergence: https://www.nexusmods.com/oblivion/mods/46220
  • FCOM_Knights: https://www.nexusmods.com/oblivion/mods/42771
  • FCOM Archery Update: http://tesalliance.org/forums/index.php?/files/file/1306-fcom-archery-update/
  • FCOM_ArcheryRealSwords (unofficial patch for FCOM Archery Update): https://www.nexusmods.com/oblivion/mods/42773

4. MMM 3.8 Performance Fix

An archive containing a performance fix for MMM 3.8 was released:

https://www.nexusmods.com/oblivion/mods/47112

Use this if you are experiencing performance issues with MMM 3.8. Be sure to also download the imp fix files.

7. Diverse Dremora Project

Diverse Dremora Project currently lacks DiverseDremoraFemales4FCOM. Some notes:

a) There are interesting informations found on the official Bethesda forums (now defunct). Preserved here:

Murandus: Just a question; Can I use the current Dremora Females ESP if I'm using OOO + MMM?

Corepc: Yes, it should work if you load it after OOO.esp/MMMforOOO.esp/ and make sure to use BASH select DDPFemales under merge patches, actors, npc faces, races only!! but some of the MMM Higher Ranks of Dremora will not be female, (Ravkyn - Solkyn Telkyn) - When OOO 1.35, MMM 3.8, comes out I will release the updated DremoraFemales plugins that cover the Higher Ranks.

b) I don't know if you are in touch with any member of Team Alpha, but it would be cool if the full plugin could be released.

10. BAIN Wizard and Optional Plugin Guidance

After reading the Expanded Load Order website, it seems that adding FCOM optional plugins such as Additional NPC Vars.esp (from Martigen) is actually not recommended.

It would be cool to actually have a definitive answer on which plugins are recommended for FCOM and which are not: either a BAIN wizard for Martigen or clearer instructions on recommended plugins for FCOM would certainly help on this regard.

11. LOOT Masterlist Cleaning Notes

According to the LOOT Masterlist:

a) The following Martigen plugins need UDR or/and ITM cleaning:

  • Mart's Monster Mod.esm
  • Martigen Shivering Isles
  • More Wilderness Life + More Wilderness Life No Gates

Do you think that's true? BOSS tells to never clean this stuff.

b) Even FCOM_CONVERGENCE.esp gets mentioned in the masterlist, where it is said:

Do not clean ITM records, they are intentional and required for the mod to function. It is safe to undelete records, but do not do anything other than that.

12. FCOM_SlofRobeTrader Dead Link

FCOM_SlofRobeTrader is based on a version 1.4 of Slof. The link contained in the optional add-ons brings you to https://www.nexusmods.com/oblivion/mods/24320, a dead end.

13. Arthmoor Optimized Francesco Resources

The FCOM Convergence page on TES Nexus makes a reference to Arthmoor optimized Francesco resources.

I don't think that part of the instructions is needed anymore. I compared the CRC of these resources and Francesco v5 (found here: https://www.nexusmods.com/oblivion/mods/40190) and they seem to match.
