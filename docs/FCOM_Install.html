First Things First

Clean Install

It's a good idea to install FCOM for the first time on a clean install of Oblivion with no other mods installed. This way, you can avoid CTDs due to incompatibility with other mods or a wrong load order or a dozen other things that could go wrong if you install FCOM on top of other mods. Play it safe, save yourself a headache, and have a nice, new and clean install of Oblivion before you begin.

Patch 1.2.0416

Make sure your game is updated to the latest version which is 1.2.0416. There are two versions of this patch - one for users who have "Shivering Isles" installed and one for those who don't. Download and install the correct patch based on your setup of Oblivion.

If you don't have the correct version, the game will CTD on launch.

FCOM Requirements

Note: For an in-depth version of this guide, please visit the FCOM site and read the Installation section.

Wrye Bash

  • Download and install Wrye Bash.

Oscuro's Oblivion Overhaul

  • Download and install OOO 137 or later. During install of OOO 136 Complete BSA Installer EXE, be sure to select the "Full" version and ignore the extra options. See the OOO Guide for details
  • Download OOO 136 Complete BSA Installer EXE from NexusMods or GitHub
  • OOO 137 Patch from NexusMods or GitHub
  • At this point, you should have the 'Meshes,' Sound,' and 'Textures' folders from OOO in your "Data" folder together with Oscuro's_Oblivion_Overhaul.esm and Oscuro's_Oblivion_Overhaul.esp
  • Activate (check) the plugins in Wrye Bash and use the FCOM Expanded Load Order page as reference and/or use the handy BOSS to sort your load order.

Martigen's Monster Mod

  • Download both files from MMM 3.8 and extract them into your "Data" folder. Do NOT use any of the ESMs/ESPs from earlier versions!
  • At this point, you should have only ONE plugin from MMM installed - Mart's Monster Mod.esm -- if you ended up with more than this, remove the others.
  • If you want, you can also use any MMM add-ons (scroll down to the General Notes section to find out more info about each add-on) like these:
    • Mart's Monster Mod - Looting NPCs & Creatures.esp
    • Mart's Monster Mod - Hunting & Crafting.esp
    • Mart's Monster Mod - Less Bone Loot.esp
    • Mart's Monster Mod - Vindasel.esp
    • Mart's Monster Mod - Resized Races.esp
    • Mart's Monster Mod - Dungeons of MMM.esp
  • Activate (check) the plugins in Wrye Bash and use the FCOM Expanded Load Order page as reference

Francesco's Leveled Creatures-Items

  • Download and install Fran's Main Package and go through the installation process (do NOT install any other component or file unless stated below):
    • 'New Items Add-On'
    • BSA File Registration
    • Shivering Isles Version (only if you have SI installed)
    • Lyrondor's Combat Behavior (recommended)
    • MOBS (highly recommended)
    • Hardcore Loot (recommended)
    • Living Economy [do NOT use the stand-alone version at the same time] (highly recommended)
    • 'Optional Leveled Guards' (for the SI version)
    • 'Optional Chance of More Enemies' (for the SI version)
    • 'Optional Chance of Stronger Enemies' (required)
    • 'Optional Chance of Stronger Bosses' (required)
    • 'New Items Add-On' [requires the FCOM add-on FCOM_FrancescosItemsAddOn.esp]
    • Merge Optional Files (do NOT merge the files!)
    • Cell Respawn Time (if you want to change this, do so in Wrye Bash)
    • Day Length Rescale (if you want to change this [I use a value of 12], do so in the console by typing the command set timescale to <number>)
    • Skill Progression Speed (use another mod like nGCD + Progress instead of this or check out other alternatives)
  • At this point, only these plugins from Fran's should be in your "Data" folder (together with 'FraNewItems.bsa'):
    • Francesco's Leveled Creatures-Items Mod.esm
    • Francesco's Optional New Items Add-On.esm
    • Francesco's Optional Chance of Stronger Bosses.esp
    • Francesco's Optional Chance of Stronger Enemies.esp
    • Francesco's Optional Chance of More Enemies.esp (only if you have SI installed)
    • Francesco's Optional Leveled Guards.esp (only if you have SI installed)
  • Activate (check) the plugins in Wrye Bash and use the FCOM Expanded Load Order page as reference

Note: this section may be outdated. If you picked the "Shivering Isles" version, you should also install the USIP 1.4 and then replace the normal USIP plugin with Fran's MOBS version of USIP, which is available from the Unofficial Fran Patches. The file is listed as USIP_MOBS_1-4. However, that version includes the Amber and Madness Ore changes from Fran's Optional Leveled Quests, so you also need Dewshine's SI-only version of Fran's Optional Leveled Quests, which is compatible with FCOM. That file is listed as Frans Optional Leveled Quests-SI Only. Alternately, you can instead use USIP_MOBS_FCOM_1-4, which does not require Frans Optional Leveled Quests-SI Only, but you'll miss out on Fran's Amber and Madness ore overhauls. Don't forget: to use either of these, you first need to install the complete USIP 1.4, then replace the USIP plugin with the MOBS version of USIP you picked.

FCOM Add-Ons & Other Supported Mods

Note: For the simplicity of this guide, I will only list examples of add-ons that I use. For a complete list, visit the FCOM site, FCOM Add-Ons page, and FCOM UESP Wiki.

Loth's Blunt Weapons

  • Activate (check) the plugins in Wrye Bash and use the FCOM Expanded Load Order page as reference and/or use the handy BOSS to sort your load order.

Unofficial Level 30+ Francesco's Leveled Item

  • Download and install the Unofficial Level 30+ Francesco's Leveled Item mod (does not need any FCOM compatibility patch).
  • Activate (check) the plugin in Wrye Bash and use the FCOM Expanded Load Order page as reference and/or use the handy BOSS to sort your load order.

Exnem's RuneSkulls

  • Download and install Exnem's RuneSkulls.
  • Use FCOM_ExnemRuneSkulls.esp and load after ExnemRuneSkulls.esp.
  • Activate (check) the plugins in Wrye Bash and use the FCOM Expanded Load Order page as reference and/or use the handy BOSS to sort your load order.

Tamriel Travellers

  • Download and install Tamriel Travellers.
  • Use FCOM_TamrielTravelers.esp and load after TamrielTravellers4000.esp and TamrielTravellersItemsNPC.esp (don't forget to use TamrielTravellers.esm).
  • Activate (check) the plugins in Wrye Bash and use the FCOM Expanded Load Order page as reference and/or use the handy BOSS to sort your load order.

T.I.E. Integration Revised

  • Download and install T.I.E. Revised.
  • Use TIE In.esp, TIE In - FCOM Convergence.esp, and TIE In - RealSwords.esp.
  • Install any optional plugins like TIE In - ExnemRuneskulls.esp, TIE In - DiverseGuardUnity.esp, TIE In - BobsGuardUnity.esp, etc. if you are also using their respective master file(s).
  • Activate (check) the plugins in Wrye Bash and use the load order guide near the bottom of the T.I.E. Revised page as reference and/or use the handy BOSS to sort your load order.

Bashed Patch

Note: For a more detailed walkthrough, please visit the FCOM site and read the Bashed Patch section.

New Game

  • Make sure you have Bashed Patch, 0.esp in your \Oblivion\Data\ folder. If you don't have it, go to \Oblivion\Mopy\Extras\ and copy Bashed Patch, 0.esp to your \Oblivion\Data\ folder.
  • Start Wrye Bash and select your entire mod list (click on the first mod to select it and then scroll all the way down to your last mod and press+hold Shift and then select it).
  • Right-click anywhere on the selected mods and select Mark Levelers.
  • Find Bashed Patch, 0.esp (which is almost always at the very end of your load order unless you're using Deadly Reflex) and right-click on it.
  • Select Rebuild Patch.
  • Select (and check) Merge Patches and make sure all the FCOM Mergeable Mods are also checked.
  • Make sure Leveled Lists is checked and the checkbox next to Automatic is also checked. Do NOT change/move anything in the Override Delev/Relev Tags window unless you know what you're doing!
  • Enable/change any other options that you want. If you don't know what a particular option does, go to the Wrye Bash site or simply hover your mouse over an option and look at the small section at the bottom to get a short description.
  • If you're happy with the changes and have double-checked everything, click OK to build your Bashed Patch.
  • Review the changes that were made and start a new game.

Saved Game

  • Load a previously saved game and then type coc TestingHall in the console (press the ~ key).
  • Sleep/wait for your cell respawn time to pass (default is 3 days) and then sleep/wait for another extra day (just to be sure).
  • Save your game and then exit.
  • Make sure you have Bashed Patch, 0.esp in your \Oblivion\Data\ folder. If you don't have it, go to \Oblivion\Mopy\Extras\ and copy Bashed Patch, 0.esp to your \Oblivion\Data\ folder.
  • Start Wrye Bash and select your entire mod list (click on the first mod to select it and then scroll all the way down to your last mod and press+hold Shift and then select it).
  • Right-click anywhere on the selected mods and select Mark Levelers.
  • Find Bashed Patch, 0.esp (which is almost always at the very end of your load order unless you're using Deadly Reflex) and right-click on it.
  • Select Rebuild Patch.
  • Select (and check) Merge Patches and make sure all the FCOM Mergeable Mods are also checked.
  • Make sure Leveled Lists is checked and the checkbox next to Automatic is also checked. Do NOT change/move anything in the Override Delev/Relev Tags window unless you know what you're doing!
  • Enable/change any other options that you want. If you don't know what a particular option does, go to the Wrye Bash site or simply hover your mouse over an option and look at the small section at the bottom to get a short description.
  • If you're happy with the changes and have double-checked everything, click OK to build your Bashed Patch.
  • Review the changes that were made.
  • Click the Saves tab in Wrye Bash.
  • Right-click on your last saved game (the one in the 'Testing Hall') and select Update NPC Levels.
  • Launch the game and load your updated saved game (to get out of the 'Testing Hall,' type coc ICMarketDistrictRindirsStaffs in the console. Or, if you prefer another location, use any of the location ID names listed on this page).
